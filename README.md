# TkoolSync
```
$ tksync
```
動かす際には`config/user-settings.json`を以下のように作成してください。  
(対話で作成するものを今作っています)
```
{
  "username":"",
  "email":"",
  "AccessToken":"",
  "remote-repository":"https://..."
}
```

進捗→https://github.com/wal-todays/TkoolSync/projects
